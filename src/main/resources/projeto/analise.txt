Aplicativo para Lojista/Prestador de Serviço X cliente

Funcionalidades

Cliente -> {
    Perfil, dados pessoais
    Buscar produtos
    Buscar serviços
    Buscar empresas/pessoas
    Cadastrar interesses (tipo um leilão)
}

Fornecedor/Prestador -> {
    Perfil, dados da empresa/prestador
    cadastrar produtos
    cadastrar serviços
    ver leiloes/necessidades de pessoas
}
